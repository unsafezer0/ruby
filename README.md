# Ruby
![Ruby](https://img.shields.io/badge/ruby-%23CC342D.svg?style=for-the-badge&logo=ruby&logoColor=white)

## Fundamentals

- [hello_world.rb](hello_world.rb) — Basic HelloWorld class with simple output.
- [var_data.rb](var_data.rb) — Variables and data types (strings, integers, booleans, floats, arrays, hashes) with type checking.
- [strings_rb.rb](strings_rb.rb) — String methods: length, case conversion (upcase, downcase, capitalize), search, split, reverse, and type conversion.
- [user_input.rb](user_input.rb) — Reading user input with type conversion (strings and integers).

## Collections

- [arrays_rb.rb](arrays_rb.rb) — Array operations: creating, accessing, looping (for, each_with_index), adding (push, unshift, <<), removing (pop, shift, delete), and utility methods (length, reverse, sort, include?, empty?, count).
- [hashes_rb.rb](hashes_rb.rb) — Hash operations: creating, adding key-value pairs, accessing values, length, iteration, key/value checking, and deletion.

## Control Flow

- [conditional_rb.rb](conditional_rb.rb) — Conditional statements: if, if-else, and if-elsif-else blocks with boolean conditions.
- [loops_rb.rb](loops_rb.rb) — Loop types: for, while, until, each, times, each_with_index, and each_index.

## Functions

- [methods_rb.rb](methods_rb.rb) — Method definition with parameters and return values.

## Error Handling & File I/O

- [err_handle.rb](err_handle.rb) — Exception handling with begin-rescue blocks (ZeroDivisionError example).
- [file_handle.rb](file_handle.rb) — File operations: writing to a file ('w' mode) and reading from a file ('r' mode) with line iteration.

## Examples & Practice

### examples/

- [examples/calculator.rb](examples/calculator.rb) — Calculator class with add, sub, mult, and div methods demonstrating object-oriented programming.
- [examples/remove_dup.rb](examples/remove_dup.rb) — Remove duplicates from arrays using both manual iteration and the built-in uniq method.
